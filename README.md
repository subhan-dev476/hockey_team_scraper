# 🏒 Hockey Team Data Scraper (Playwright)

A professional Python web scraping project that extracts hockey team statistics from a website using Playwright.

---

## 📌 Project Overview

This scraper collects structured hockey team data including:

- Team Name  
- Year  
- Wins  
- Losses  
- Win Percentage  

The script automatically navigates through multiple pages and extracts all available data efficiently.

---

## 🚀 Features

- Multi-page scraping (pagination support)  
- Clean and structured data extraction  
- Error handling using logging  
- Automatic CSV file generation  
- Fast and efficient scraping using Playwright  

---

## 🛠️ Tech Stack

- Python  
- Playwright  
- CSV  
- Logging  

---

## 📦 Installation Guide

### 1. Clone Repository

git clone https://github.com/yourusername/hockey-team-scraper.git  
cd hockey-team-scraper  

---

### 2. Install Dependencies

pip install -r requirements.txt  

---

### 3. Install Playwright Browsers

playwright install  

---

## ▶️ How to Run

python hockey_team_scraper.py  

---

## 📁 Output

The script generates a CSV file like:

Hockey_Team_Data_YYYY_MM_DD.csv  

This file contains all scraped hockey team data in a structured format.
